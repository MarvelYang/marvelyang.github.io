# Object.create

原理

实质上是把传入的参数对象作为新对象的原型

  1. 创建了一个构造函数
  2. 把构造函数的原型prototype指向传入的参数对象
  3. 返回新建构造函数对象的实例

代码

// 手动实现Object.create
const create = obj => {
    function F() {}
    F.prototype = obj;
    return new F();
};

const obj = { a: 1 };
const obj_new = Object.create(obj);
obj.a = 2;
console.log('obj', obj, obj_new, obj_new.a);
// 'obj', {a: 2}, {}, 2
