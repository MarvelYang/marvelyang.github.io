# let、const

# 定义

let和const是 ES6 新增的两个变量声明语句，其作用是声明一个块级作用域的本地变量。

TIP

关于 let, const, var 三者的不同

  • var语句声明的变量是全局或者整个函数块（函数作用域），let和const语句是块级作用域。

  • Javascript 引擎在编译阶段会把var声明的变量提升到 作用域顶部，把let和const声明的变量放到 TDZ (Temporal Dead Zone 临时死区) 。

  • var和let可以重复赋值，const 不能重复赋值 (不能改变绑定，但是可以改变值，例如对象，数组) 。
