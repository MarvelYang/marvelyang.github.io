# 20. Valid Parentheses (有效括号)

# 问题

给定一个只包括 '('，')'，'{'，'}'，'['，']'  的字符串，判断字符串是否有效。

有效字符串需满足：

  • 左括号必须用相同类型的右括号闭合。
  • 左括号必须以正确的顺序闭合。
  • 注意空字符串可被认为是有效字符串。

示例

  • 示例 1

    输入："()"

    输出：true

  • 示例 2

    输入："()[]{}"

    输出：true

  • 示例 3

    输入："(]"

    输出：false

  • 示例 4

    输入："([)]"

    输出：false

  • 示例 5

    输入："{[]}"

    输出：true

TIP

# 栈

栈是一种常见的数据结构，其特点是“后进先出”，详细概念可以查询相关资料。

想象一下，每次读取一个字符会有几种情况？

  • 第一种情况：空字符串，直接返回 false

  • 第二种情况：是一个左括号，保存在栈顶，继续读取

  • 第三种情况：是一个右括号，那么这个时候就会有三个操作

    1. 查看刚刚保存的字符，看是否匹配，如果是，那么就是合法，可以把这对括号消掉（有点像连连看的感觉），也就是把栈顶元素去掉，继续下一轮。
    2. 如果不匹配，说明这里就存在了一对无效括号，那么就可以返回 false，结束。
    3. 栈里没有元素，那肯定就无效，返回 false。

时间复杂度：首先每一个元素进出栈的时间为，然后最坏情况每个元素都会进栈一次，所以最后计算出就是。

空间复杂度：最坏情况下每个元素都需要保存，所以是。

# 代码

# 栈

const isValid = s => {
    if (!s) return true; // 如果是空字符串直接返回结果

    let stack = [];
    let paren_map = { ')': '(', ']': '[', '}': '{' }; // 创建映射关系
    for (let i of s) {
        if (!paren_map[i]) {
            stack.push(i); // 如果不是右括号就进栈
        } else if (!stack.length || paren_map[i] !== stack[stack.length - 1]) {
            // 是右括号，但是栈为空或者不等于栈顶元素
            return false;
        } else {
            // 匹配栈顶元素，则删除栈顶元素
            stack.pop();
        }
    }
    return stack.length === 0;
};
