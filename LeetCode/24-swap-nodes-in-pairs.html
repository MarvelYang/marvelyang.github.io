# 24. Swap Nodes in Pairs (两两交换链表中的节点)

# 问题

给定一个链表，两两交换其中相邻的节点，并返回交换后的链表。

你不能只是单纯的改变节点内部的值，而是需要实际的进行节点交换。

示例

输入 1->2->3->4
输出 2->1->4->3
1
2

# 思路

TIP

# 方法一: 递归

参考自 LeetCode 上 灵魂画师 的递归解法

递归写法要观察本级递归的解决过程，形成抽象模型，因为递归本质就是不断重复相同的事情。而不是去思考完整的调用栈，一级又一级，无从下手。如图所示，我们应该关注一级调用小单元的情况，也就是单个。

递归中应该关注三个条件：

  • 返回值
  • 调用单元执行过程
  • 终止条件

按照上面的思路，看下本题中：

  • 返回值：交换完成的子链表
  • 调用单元：主要就就是交换 head 和 next 节点位置，把 head 连接后续交换完成了的子链，再把 next 连接 head，交换过程就结束了
  • 终止条件：head 为空，或者 next 为空（代表当前无节点或者只有一个节点无需交换）

# 方法二：迭代

TIP

这个方法就是从头开始交换：

  • 先设置一个临时的虚拟头节点 dummyHead，命名一个指针变量 p
  • 设置两个要交换的节点为 node1，node2
  • 设置 node2 的后续节点为 next

上面大概就是这个样子：dummyHead(p) -> 1(node1) -> 2(node2) -> 3(next) -> 4

循环过程的操作：

  • 将node2节点的 next 链接node1节点 // 2(node2) -> 1(node1)
  • 将node1节点的 next 链接next节点 // 1(node1) -> 3(next)
  • 将dummyHead节点的 next 链接node2节点 // dummyHead(p) -> 2(node2)
  • 结束交换 // dummyHead(p) -> 2(node2) -> 1(node1) -> 3(next) -> 4
  • 把指针 p 指向节点 node1 （节点node1 实际上可以看成是下一轮循环开始待处理的节点 3 的“虚拟头”节点）

循环结束后，返回一开始的 dummyHead 节点的 next 指向节点node2作为结果

# 代码

# 方法一: 递归

/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
const swapPairs = head => {
    // 判断是否满足退出条件
    if (head === null || head.next === null) return head;

    // 链表实例：head(node1) -> next(node2) -> after...
    let next = head.next; // 存储需要交换的node2节点

    // head.next = swapPairs(next.next); // head连接node2节点后面的交换好的子链 head(node1) -> after...
    // next.next = head; // 把node2节点连接到head节点，next(node2) -> head(node1) -> after...

    [head.next, next.next] = [swapPairs(next.next), head]; // 简化写法

    return next; // 完成交换，返回next(node2)节点
};

# 方法二：迭代

/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
const swapPairs = head => {
    let dummyHead = new ListNode(null); // 创建虚拟头节点
    dummyHead.next = head; // 链接head节点
    let p = dummyHead; // 指针p

    while (p.next !== null && p.next.next !== null) {
        let [node1, node2] = [p.next, p.next.next]; // 设置两个节点指针node1，node2

        [p.next, node2.next, node1.next] = [node2, node1, node2.next]; // 设置对应的指针指向（参考思路过程）

        p = node1; // 改变p指针位置，指向交换后node1节点的位置，作为下一轮循环的起始
    }

    return dummyHead.next;
};
