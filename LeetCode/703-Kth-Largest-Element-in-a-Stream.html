# 703. Kth Largest Element in a Stream (数据流中的第 K 大元素)

# 问题

设计一个找到数据流中第 K 大元素的类（class）。注意是排序后的第 K 大元素，不是第 K 个不同的元素。

你的  KthLargest  类需要一个同时接收整数  k 和整数数组 nums  的构造器，它包含数据流中的初始元素。每次调用  KthLargest.add，返回当前数据流中第 K 大的元素。

示例

int k = 3;
int[] arr = [4,5,8,2];
KthLargest kthLargest = new KthLargest(3, arr);
kthLargest.add(3);   // returns 4
kthLargest.add(5);   // returns 5
kthLargest.add(10);  // returns 5
kthLargest.add(9);   // returns 8
kthLargest.add(4);   // returns 8

# 思路

这里首先可以简化思考，如果只需要给出最大的值MAX，那就是每次只需要记录一个最大值，后续每个进来的元素和MAX作比较即可。

那么假如是第 K 个呢？

那就是返回 K 个最大值，然后从这 K 个值里面，取最小的那个去比较即可。

所以这道题就可以看作是：

  • 维护 K 个最大元素
  • 每当后续有新元素加入，就取出 K 里面最小的值比较
  • 如果新元素 > ，那就把从维护的元素里面踢出去（真 🐔 儿丢人，你退群吧）
  • 反之，则继续比较后面的数据流
  • 比较完毕都返回当前第 K 大的元素

那怎么维护这 K 个元素，就可以发挥各自的思路，比如用快速排序等。

TIP

# 最小堆

这里主要用到一个最小堆的数据结构去帮助解题，最小堆详细定义可以自行 google，简单来说就是类似一个树结构的数据结构，顶端的节点是当前所有元素的最小值，每次更改其中的元素，会进行重新排序。

Java 等语言都有实现该数据结构（其算是一种语言的标准常用数据结构的一种，直接使用就可以，不需要自己再手动实现。因为 JavaScript 没有，所以下面用 Java 去实现，然后用 JavaScript 去模拟写法，最重要是理解其中的思想即可。）

# 代码

# 最小堆

class KthLargest {
    final PriorityQueue<Integer> q;
    final int k;

    // 主线程
    public KthLargest(int k, int[] nums) {
        this.k = k;
        q = new PriorityQueue<>(k);
        for (int n : nums)
            add(n);
    }

    public int add(int val) {
        if (q.size() < k)
            q.offer(val); // 如果是还不满足 K 个，则直接填充
        else if (q.peek() < val) {
            q.poll();
            q.offer(val); // 如果数据流新的元素比堆顶（最小值）要大，那么就删除堆顶元素，并填充新的元素
        }
        return q.peek(); // 返回堆顶元素，即第 K 个最大值
    }
}

下面用 JavaScript 去模拟，实现最小堆的方法就不手动实现了，假设我们已经存在这么一个方法。

// MinPriorityQueue类，js的实现的最小堆数据结构类，
// 包含获取堆顶元素的方法peek()，获取堆大小的方法size()，删除堆顶元素方法poll()，添加元素方法offer()，
// 我们直接使用即可

/**
 * Your KthLargest object will be instantiated and called as such:
 * var obj = new KthLargest(k, nums)
 * var param_1 = obj.add(val)
 */

class KthLargest {
    constructor(k, nums) {
        this.q = new MinPriorityQueue(k);
        this.k = k;

        for (let n of nums) {
            this.add(n);
        }
    }

    add(val) {
        if (this.q.size() < k) {
            q.offer(val);
        } else if (this.q.peek() < val) {
            q.poll();
            q.offer(val);
        }

        return q.peek();
    }
}
