# 206.Reverse Linked List (反转链表)

# 问题

反转一个链表。

示例

输入: 1->2->3->4->5->NULL
输出: 5->4->3->2->1->NULL
1
2

进阶: 你可以迭代或递归地反转链表。你能否用两种方法解决这道题？

# 思路

本题思路主要有两种，迭代和递归（递归效率要低不少，后续再补上)

TIP

# 迭代

  1. 设置三个指针pre（记录前驱节点）、cur（记录当前节点）、next（记录下一个节点）

  2. 设置cur为给定的节点（也就是head）

  3. 循环过程

    1. 判断cur指向节点是否为 Null，如果是，结束循环，返回pre指向节点
    2. cur指向节点不为空，则设置next指针指向cur的下一个节点（next相当于临时存储了原来的后一个节点）
    3. 设置cur的下一个节点为pre指针指向节点
    4. 改变指针位置，pre指针指向cur指针节点、cur指针指向next指针节点
    5. 重复（1）（2）（3）（4）

  4. 返回pre指针指向节点为结果

/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var reverseList = function(head) {
    let [cur, prev] = [head, null];

    while (cur) {
        [cur.next, prev, cur] = [prev, cur, cur.next]; // ES6的结构赋值，可以少写一些代码
    }
    return prev;
};
