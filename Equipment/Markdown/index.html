# Markdown 列表

无序列表，使用星号*、加号+、减号-标记

  • *第一项

  • *第二项

  • +第一项

  • +第二项

  • -第一项

  • -第二项

有序列表，使用数字+.

  1. 第一项
  2. 第二项

列表嵌套：列表嵌套只需要子列表中的选项添加四个空格即可

  1. 第一项
    • 第一项嵌套
    • 第一项嵌套
  2. 第二项
    • 第二项嵌套
    • 第二项嵌套

# Markdown 链接

链接使用格式:

[链接名称](链接地址) 或者 <链接地址>

例如：Google 地址

直接使用链接地址：https://www.google.com

# Markdown 图片

Markdown 图片语法格式:

![alt 属性文本](图片地址)

![alt 属性文本](图片地址 "可选标题")

  • 开头一个感叹号!
  • 紧跟方括号[]，里面放上图片的替代文字
  • 最后跟一个括号()，里面放图片的网址，最后还可以用引号""输入“title”文字

当然也可以使用变量：

这个链接用 1 作为网址变量Google

Markdown 还没有办法指定图片的高度和宽度，如果你需要的话，你可以使用普通的标签

# Markdown 表格

Markdown 制作表格使用|来分隔不同的单元格，使用-来分隔表头和其他行

语法格式如下:

| 表头 | 表头 |

| ---- | --- |

| 单元格 | 单元格 |

表头 表头
单元格 单元格

对齐方式：

  • -:设置内容和标题栏居右对齐
  • :-左对齐
  • :-:居中对齐
